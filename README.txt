1. установить на ПК Node.js по ссылке https://nodejs.org/dist/v22.16.0/node-v22.16.0-x64.msi
2. перейти в папку Data и загрузить файл с остатками, переименовать его в stocks (тот файл, что кидает менеджер галыча, скрипт автоматом выберет нужный файл соответствия бязь/поплин)
3. запустить скрипт файлом run-galtex.bat
4. итоговый файл с остатками сохранится с названием galtex-ozon-stocks-updated.xlsx и galtex-wb-stocks-updated.xlsx
